# Sistema de Tickets Angular

Aplicación web desarrollada en Angular para la gestión de tickets de soporte.

## 🚀 Funcionalidades

- Asignación de tickets
- Filtros por categoría y prioridad
- Visualización de detalle de ticket
- Cierre de tickets

## 🛠️ Tecnologías

- Angular
- TypeScript
- Bootstrap
- FontAwesome
